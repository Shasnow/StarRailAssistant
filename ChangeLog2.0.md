# SRA已更新到 2.0
## Full Changelog：
- 更新日期：
- 版本：2.1
- 主要更新内容：
  - 新增对 货币战争·零和博弈 的自动化支持
  - 新增 历战余响·铁骸的锈冢、凝滞虚影·明辉日珥
- 功能调整：
  - 命令行子命令 run 的简短选项`-c`被确认存在冲突，请使用完整选项`--config`代替
  - 操控面板的启动按钮现在会保存上一次的模式选择
  - 差分宇宙和货币战争现在整合到Cosmic Strife 旷宇纷争 (旧称 Simulated Universe 模拟宇宙)页面中
- 问题修复：
  - 修复了凝滞虚影的一处文本错误
- 测试版说明：
  - 本版本为测试版，如果在使用过程中遇到任何问题，请及时反馈
  - 货币战争中 标准博弈/超频博弈 的模式选择框不可用，仅作占位使用。目前固定为 标准博弈 模式
- 货币战争自动化使用说明：
  - 常规开始：令游戏处于大世界界面或货币战争主界面，打开SRA，在任务 Cosmic Strife 标签页勾选此任务，并勾选货币战争，选择执行次数，点击开始
  - 接管对局：在货币战争对局备战阶段，将场上所有角色放回备战席，确保商店为收起状态，打开SRA，选择 Cosmic Strife 标签页，勾选 货币战争 选项，选择执行次数，点击开始
