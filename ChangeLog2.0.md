# SRA已更新到 2.0
## Full Changelog：
- 更新日期：
- 版本：2.2
- 主要更新内容：
  - 优化包体大小：现在SRA使用便携版Python运行时，整体包体积大幅缩减。推荐重新下载完整安装包以获得最佳体验。
  - 新增手动检查更新按钮。
  - 新增重启控制台按钮。
  - 已支持热更新，当某次更新为热更新时，更新完成后无需重启SRA，只需要重启控制台即可。
- 功能调整：
  - none
- 问题修复：
  - none
- Python 环境说明: 
  - python下载缓慢：
  - 如果下载缓慢，请先忽略首次运行的错误提示，在设置页面中更改下载链接后重新检查并修复Python环境。
  - 或手动下载便携版Python运行时并放置在SRA根目录下的`python`文件夹（如果没有此文件夹，请新建）中。
  - get-pip.py 下载缓慢：
  - 如果下载缓慢，请手动在网络上下载get-pip.py并放置在SRA根目录下的`python`文件夹（如果没有此文件夹，请新建）中，然后重新检查并修复Python环境。
  - pip 下载缓慢：
  - 前往设置页面设置pip镜像源，然后重新检查并修复Python环境。
