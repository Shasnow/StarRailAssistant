# SRA已更新到 2.0
## Full Changelog：
- 更新日期：
- 版本：2.0
- 主要更新内容：
  - 我们已将 UI 从`PySide6`迁移到`Avalonia`, 希望这能为您带来更现代化和更舒适的用户体验。
  - 现在运行SRA需要安装`.NET 8.0`。如果您未安装，您会在首次运行时收到提示，请按照提示进行安装。
  - SRA更新器已集成到主程序中，您可以通过主界面直接检查和安装更新。
  - SRA现在提供了一个命令行界面（CLI），允许高级用户跳过图形化用户界面（GUI）运行SRA，但仍需GUI进行任务配置。
  - 为第三方工具调用做好了准备。
- 功能调整：
  - 设置->任务通用设置->线程安全：此选项已经过时，修改它不会有实际效果。SRA现在使用进程来执行任务，以便随时终止。
  - 设置->常规->开机自启：此选项已经过时，修改它不会有实际效果。SRA需要管理员权限，无法开机自启，请使用系统的计划任务。
  - 已取消配置方案数量上限，现在您可以创建任意数量的配置方案。您可能需要手动清理不再使用的配置方案的文件。
  - 原全局配置文件`globals.json`已被弃用，它被拆分为`cache.json`和`settings.json`，分别用于缓存数据和存储应用设置。
  - 配置方案文件格式已更改，旧版本配置方案将全部失效。
  - SRA相关文件的存储路径已更改，已经转移到`C:/Users/您的用户名/AppData/Roaming/SRA`目录下。
- 尚未实现：
  - 设置->高级->检查文件完整性：此功能尚未实现，未来版本中可能会添加。
  - 插件系统：插件系统尚未实现，未来版本中可能会添加。
  - 多语言支持：目前仅支持简体中文，未来版本中可能会添加其他语言支持。
  - 定时执行任务：此功能尚未实现，未来版本中可能会添加。
  - 全局热键支持：此功能尚未实现，未来版本中可能会添加。
  - 多分辨率支持：此功能尚未实现，未来版本中可能会添加。
