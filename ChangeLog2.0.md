# SRA已更新到 2.x
## Full Changelog：
- 更新日期：
- 版本：2.10

### 主要更新内容:
- 开发者模式之 -- 使用Python从源代码运行后端。

### 功能调整：
- 货币战争刷开局中当第一位面未匹配到投资策略需要重开时，直接重开，不进行选择和确定。
- 移除了差分宇宙刷单层的“自动战斗”检查以提升效率。因此建议在游戏设置中将“是否沿用自动战斗设置”设置为“是”并保持自动战斗开启，以确保在刷单层时能够正确使用自动战斗。
- 游戏路径检测现在将检测设备上所有的游戏路径，并以列表的形式呈现给用户选择，以适应多游戏路径的情况。如果无法检测到您的游戏路径，考虑关闭自动检测并手动添加游戏路径。

### 问题修复：
- 修复了当启用领取兑换码奖励时，无法正常领取邮件奖励导致后续流程异常的问题。
- 修复了多个配置运行之间会有判定游戏已经在运行但实际并没有任何游戏启动的问题。
- 修复了差分宇宙刷单层中角色容易走太前导致无法正常战斗的问题。
- 修复了货币战争中无法启用超频博弈的问题。
- 修复了启动游戏任务无法单独指定游戏路径的问题。

#### 开发者模式：
- 快速多次点击版本号可以进入开发者模式
- 开发者模式下可以选择使用Python从源代码运行后端，便于开发者进行调试和测试。
- 使用此功能需要选择一个Python解释器（可以是虚拟环境中的解释器），并确保已安装所需的依赖项。
- 选择Python解释器后，程序将使用该解释器从源代码运行后端，而不是使用预编译的二进制文件。这对于开发者来说可以更方便地进行调试和修改代码。
- 作为普通用户，如果您熟悉Python环境和Git版本控制，可以使用此方法来运行最新的、未发布的代码版本，以体验最新的功能和修复。
