### 主要更新内容:
- 适配4.2版本「如是，众生欢笑不已」
  - 新增拟造花萼（赤）- 法吉娜之心（虚无）
  - 货币战争新增角色“不死徒”、“绯英”、“银狼LV.999”
  - 货币战争同步“飞霄”、“长夜月”的费用更改

### 功能调整：
- 货币战争攻略模板语义调整，"角色名称": 数量 -> "角色名称": 星级。
- 货币战争现在只会给在攻略中的角色穿戴装备。
- 后端启动参数调整，现在`-e`参数无需使用引号包裹命令。使用`+`替代`&`来连接多个命令。
- 饰品提取暂不支持使用支援角色。

### 问题修复：
- 修复了在星际和平指南中点击差分宇宙前往参与时未直接进入差分宇宙界面导致无法进行差分宇宙任务的问题。
- 修复了任务结束后无法正确登出账号，导致多账号流程切号失败的问题。
- 修复了差分宇宙中当只有一张面具可选时，无法正确选择面具的问题。
- 修复了差分宇宙中无法处理惊世奇迹的问题。
- 修复了饰品提取当队伍中有空位时无法开战的问题。
- 修复了货币战争中触发“我来当策划”事件后卡住的问题。

### 更新说明：
#### -e参数示例：
- 阻塞式运行默认配置并在完成后退出：`-e run Default + exit`
- 阻塞式运行默认配置并在完成后进入交互式命令行：`-e run Default`
- 非阻塞式运行默认配置：`-e task run Default`
